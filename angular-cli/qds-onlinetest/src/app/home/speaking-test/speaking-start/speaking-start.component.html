SPEAKING TEST
  • 1. The time limit for this test can be found at the top right corner.
  • 2. Make sure you have: good internet, quiet place, headphones.
    Press “Mic Check” button below to test your microphone.
  • 3. Tap the microphone and record the following 15 sentences.
    You can record each sentence twice.
